# Foodie base project

You can extend your project base on this, which contains the local jar - `[foodie-core-2.0.jar](src%2Fmain%2Fresources%2Ffoodie-core-2.0.jar)`

### How to run
```shell
mvn install
mvn clean compile exec:java
```

expect as see
```shell
Order ID = 1
Customer ID = 1
Hello world!
```
